# SUNOSAVE
Save lyrics and prompts for SUNO.AI


## Features
- Save prompts
- Save lyrics


## Views
- Dashboard
- Save prompt form

## Form
- Title - *required*
- Prompt - *required*
- Lyrics - *optional*

## Data
```
const SUNOSAVE = {
  id: number,
  title: string,
  prompt: string,
  lyrics: string,
  date: date,
}
```

## Tech Stack
- Prisma ORM
- Postgres

- React
- Express
